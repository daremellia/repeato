# 💡 [Idea] 첫 번째 자동화 스크립트 기획: 회의록 자동 요약 및 정리

**작성일자:** 2026년 6월 11일
**작성자:** 리피토 프로젝트 관리자

## 1. 해결하려는 문제 (Problem)
매번 회의가 끝난 후 텍스트로 적힌 난잡한 회의록을 '목적, 주요 안건, 결론, 다음 할 일(To-do)' 포맷으로 다시 다듬는 데 30분 이상의 시간이 소모된다. 이 반복적인 문서 정리 작업을 자동화하고 싶다.

## 2. 작동 방식 (How it works)
1. **입력 (Input):** 다듬어지지 않은 원본 회의록 텍스트(예: `raw_meeting.txt`)를 지정된 폴더에 넣는다.
2. **처리 (Process):** 스크립트가 텍스트를 읽어 들인 후, 정해진 템플릿(포맷)에 맞게 텍스트를 재배치한다.
3. **출력 (Output):** 깔끔하게 정리된 마크다운 파일(`formatted_meeting.md`)이 자동으로 생성된다.

## 3. OpenAI API 활용 계획 (Next Step)
단순 텍스트 재배치를 넘어, 향후 OpenAI API를 연동하여 다음 기능을 추가할 예정이다.
- 긴 회의 대화 내용을 AI가 세 줄로 요약
- 회의 내용 중 '누가, 언제까지, 무엇을 할 것인지(Action Item)'를 AI가 자동으로 추출하여 체크리스트 생성
