# 📚 TIL (Today I Learned) - AI & Tech Note

## 📌 이 프로젝트는 무엇인가요?
매일 새롭게 배우고 깨달은 기술적인 지식과 AI 활용법을 기록하는 개인 학습 아카이브이자 오픈소스 저장소입니다. 

## 💡 왜 만들었나요?
빠르게 변화하는 AI 기술과 업무 자동화 지식을 머릿속에만 두지 않고, 체계적으로 정리하여 필요할 때마다 꺼내 쓰기 위해 시작했습니다. 또한, 저와 비슷한 고민을 하며 독학하는 다른 크리에이터나 학습자들에게도 도움이 되는 공개 자산이 되기를 바라는 마음으로 오픈소스로 공유합니다.

## ✨ 어떤 내용이 담기나요?
- **AI & LLM:** OpenAI API 활용법, 프롬프트 엔지니어링 팁, 다양한 AI 툴 실전 적용기
- **Tech & Git:** 기초 코딩(Python 등), Git/GitHub 활용법, 에러 해결(Troubleshooting) 과정
- **Workflow:** 영상 편집 및 업무 프로세스를 효율화하는 디지털 워크플로우 연구

## 🚀 어떻게 활용하나요?
1. 상단의 카테고리(폴더)별로 분류된 마크다운(`.md`) 문서를 클릭합니다.
2. 날짜별 또는 주제별로 정리된 핵심 요약과 공부 내용을 확인합니다.
3. 내용 중 수정이 필요하거나 더 좋은 방법이 있다면 언제든 Issue나 Pull Request를 통해 의견을 남겨주세요.

## 🌱 앞으로의 계획
- **지속적인 기록:** 매주 최소 2~3회 이상 새로운 학습 내용을 꾸준히 업데이트합니다.
- **트러블슈팅 세분화:** AI 툴이나 프로그램을 사용하면서 겪은 시행착오와 해결 과정을 상세히 기록할 예정입니다.
- **오픈소스 기여:** 다른 사람들의 피드백을 적극 수용하여 누구나 읽기 쉬운 지식 저장소로 발전시키겠습니다.
